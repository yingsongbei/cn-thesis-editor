# cn-thesis-editor

面向植物农业方向和更广泛生命科学领域的中文博士学位论文语言编辑 skill。

## 适用场景

- 中文摘要、英文摘要
- 绪论和文献综述
- 材料与方法
- 结果
- 讨论
- 结论、创新点、不足与展望
- 参考文献和文内引用格式检查

## 默认原则

- 默认润色强度：中度
- 默认风格：中国顶尖大学优秀博士论文风格
- 学科倾向：植物农业方向，同时覆盖生命科学
- 允许拆分长句和调整段落顺序
- 更稳、更保守、更少主观判断
- 不允许新增事实、数据、机制、引用或参考文献信息
- 每次输出风险提示

## 润色强度

轻度：只修改病句、错别字、标点、不顺和轻微口语化表达，基本保留原句结构。

中度：优化清晰度、逻辑衔接、段落流动、术语一致性和学术表达；可以拆分长句，必要时调整句序或段落顺序。

重度：重组段落、合并重复内容、强化章节逻辑和论证层次；仍然不得改变科学含义，并需要标出所有可能影响含义的修改。

## 推荐用法

```text
请使用 cn-thesis-editor 对下面这段博士论文讨论部分进行中度润色，保持科学含义不变，并给出风险提示。
```

```text
请使用 cn-thesis-editor 修改英文摘要，不需要逐字对应中文摘要，但要保持科学事实一致，语言逻辑接近 CNS 顶刊摘要风格。
```

```text
请使用 cn-thesis-editor 检查这段参考文献是否符合学校要求；如果没有学校要求，请按 GB/T 7714-2015 检查。
```

## 参考文献规则

学校或学院格式要求优先。没有学校要求时，默认按 GB/T 7714-2015 检查参考文献。GB/T 7714-2025 已发布，计划于 2026-07-01 实施；若学校明确采用 GB/T 7714-2025，则按 2025 版执行。

skill 不会虚构 DOI、页码、作者、年份、期刊名、网址或其他缺失字段，只会标出需要你核对的位置。
